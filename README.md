> **🚀 Git 컨벤션**
  
| 커밋 유형 | 설명 |
|-----------|------------------|
| `feat`    | 새로운 기능 추가 |
| `fix`     | 버그 수정 |
| `docs`    | 문서 수정 |
| `style`   | 코드 포맷팅, 세미콜론 누락 등 (코드 변경 없음) |
| `refactor`| 코드 리팩토링 |
| `test`    | 테스트 코드 추가 및 수정 |
| `chore`   | 빌드 업무 수정, 패키지 매니저 수정 |
| `merge`   | 브랜치 merge 및 pull |
